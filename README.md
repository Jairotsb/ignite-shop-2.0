## Tecnologias Utilizadas
- ReactJs

## Passo a passo

1. Clone o repositório
```
git clone 
```
2. Instale as dependências
   
```
yarn / npm install
```

3. Rode em ambiente de desenvolvimento.
```
npm run dev
```
